# FactoryVoice — Agent Onboarding & MVP Setup Process
**Version 1.0 (Web-Chat Model)**

*This document describes how a FactoryVoice field agent sets up the AI assistant on-site at a manufacturing client's premises. The owner does nothing technical — the agent handles everything in 30 minutes, bypassing complex API approvals by using a smart web-link strategy.*

# **1. The Concept**
Most small manufacturers are not comfortable with technology. Asking them to fill forms, connect APIs, or approve prompts on a computer is a barrier that will cause drop-offs. Our solution is simple: we send a trained field agent to the factory.

The agent visits the owner, takes a factory tour, talks to the team, and fills everything into the FactoryVoice Agent App on their phone. Instead of waiting weeks for Meta/Facebook API approvals, the bot goes live instantly via a dedicated FactoryVoice web chat, linked directly from the owner’s standard WhatsApp.

**THE OWNER'S EXPERIENCE**
- Agent arrives. Owner shows them around for 20 minutes.
- Agent asks a few questions and types answers into the Agent App.
- Agent shows the owner 3 sample replies generated by the AI.
- Owner approves. 
- Agent takes the owner's phone and turns on the standard WhatsApp "Auto-Reply" feature, pasting a unique FactoryVoice web link (e.g., `factoryvoice.app/chat/sharma-steels`).
- Bot is live. Owner gets a short demo of their Dashboard to track leads.

**RESULT**
The owner touched no form, connected no API, and gave no passwords. The agent did everything. The bot is live the same day.

# **2. What the Agent Collects**
The agent collects information across six areas. All collected through the Agent App — a simple mobile-friendly web interface.

*(Note: Data collected remains the same: Business Basics, Products, Pricing, Customers, Lead Time & Capacity, Contact Info).*

# **3. The Agent App (Flow)**
The Agent App is a mobile web app used strictly by the agent. It guides them through data collection one question at a time.

**APP SCREENS — FLOW**

|**Screen**|**Content**|
| :- | :- |
|Login|Agent email + PIN.|
|New visit|Enter: client name, city. Creates a new onboarding record.|
|Sections 1 to 6|Agent types factory details (Products, Prices, Lead times, etc.) while talking to owner.|
|Generate brain|Button: 'Generate Bot Profile'. System sends data to AI and builds the instruction manual.|
|Review samples|3 sample Q&A pairs shown on screen. Owner approves or agent edits.|
|**Setup Auto-Reply**|**Agent is given the factory's unique bot link. Agent manually pastes this into the owner's WhatsApp Business app as a greeting message.**|
|Go live & Dashboard|Bot is activated. Agent logs the owner into the Dashboard to view future leads.|

# **4. How Agent Data Becomes the Bot's Brain & Chat Flow**

**STEP 1 — DATA SAVED & BRAIN GENERATED**
The system takes the agent's notes and builds the AI System Prompt (the rulebook). Example:
*"You are the assistant for Sharma Steel Fabricators. MS sections are Rs 65–75/kg. Minimum order is 1 tonne. Tell customers to contact Mr. Sharma for exact quotes."*

**STEP 2 — THE BOT GOES LIVE (THE AUTO-REPLY HACK)**
Instead of complex Meta API connections, the agent sets up a simple WhatsApp Auto-Reply on the owner's phone:
*"Hi! Welcome to Sharma Steels. Our AI assistant can instantly answer your questions about stock, prices, and lead times. Click here to chat: `factoryvoice.app/chat/sharma-steels`"*

**STEP 3 — THE CUSTOMER EXPERIENCE (LEAD CAPTURE)**
When a buyer clicks the link, the following happens:
1. They land on a mobile web page that looks exactly like a WhatsApp chat.
2. **Lead Gate:** The bot says, *"Welcome! Please enter your WhatsApp number so Mr. Sharma can follow up with your quotes."*
3. The buyer enters their number (saving them as a Lead in the database).
4. The buyer chats with the Claude AI bot about products and prices.

# **6. Updating the Bot After Setup**
Because we are using a Web-Chat model for Version 1.0, the bot is updated in two ways:

**Option A — Owner updates via dashboard**
The owner logs into the FactoryVoice Dashboard on their phone/computer and edits their business profile (e.g., changes the price of steel). The bot learns this instantly.

**Option B — Agent return visit**
If the business changes significantly, the agent does a brief 15-minute return visit, edits the profile in the Agent App, and regenerates the brain.

# **7. System Components — What We Are Building**
There are three distinct interfaces in the FactoryVoice ecosystem.

|**Component**|**Who Uses It**|**What it does**|
| :- | :- | :- |
|**1. The Agent App**|Field Agent|Mobile web app. Used during the factory visit to collect data and generate the AI bot profile.|
|**2. Customer Web Chat**|The Buyers|A clean, WhatsApp-style web page where buyers type their phone number (lead capture) and chat with the AI.|
|**3. Owner Dashboard**|Factory Owner|A web portal where the owner logs in to see all captured phone numbers, read the chat logs, and update their prices.|

# **8. What the Bot Knows After Setup**
After a successful agent visit, the bot knows everything needed to handle 80–90% of common customer enquiries:
- **Product knowledge:** Materials, grades, standard stock vs custom work.
- **Commercial:** Price ranges, MOQ, delivery areas.
- **Operational:** Lead times, production capacity.

*If the customer asks for something highly specific (like a custom CAD drawing review), the bot is programmed to say: "Please send your drawing directly to this WhatsApp chat, and the owner will reply shortly."*

# **9. Agent Training Requirements**
Agents do not need a technical background. 

**Core Training Additions for Web-Link Model:**
- Agents must be trained on how to navigate the native "WhatsApp Business" app on Android and iOS.
- They must know exactly how to set up, turn on, and format a "Greeting Message" with a URL link. 
- They must know how to train the owner on using the Owner Dashboard to check their daily leads.

# **10. Strategic Advantages of the V1.0 Web-Link Model**

|**Feature**|**Why we are doing it this way**|
| :- | :- |
|**No Meta API Red Tape**|We do not have to wait 3–4 weeks for Facebook business verification, App Reviews, or API limits. We can launch tomorrow.|
|**Guaranteed Lead Capture**|By forcing the customer to type their number before the chat begins, we generate a high-quality "Lead List" for the factory owner.|
|**Zero API Messaging Costs**|Because the chat happens on our website instead of WhatsApp, we pay $0 to Meta. We only pay fractions of a cent for the AI text generation, resulting in massive profit margins.|
